Como baixar os jogos

Siga estes passos simples:

1) Clique em um jogo para abrir os detalhes;
2) Use o botão "Download" para abrir o link do arquivo em nova guia;
3) Extraia ou instale conforme as instruções do arquivo. Se precisar de ajuda, assista o vídeo ou use o link de Suporte no menu.
